### ✔️BaseType : 숫자형, 문자열 자료형
* 문자열 길이구하는 함수 : len(string)
* 예약어 출력 : 두번 연속 작성(%%, {{}}...)

### ✔️ListType : 리스트 자료형
1. create
   * a = []  / a = list()
1. read
   * sort()/reverse() : 오름차순/내림차순(리스트 요소를 정렬)
1. update
   * .append() : 이미 만들어진 리스트에 객체를 추가(뒤에 추가)
   * .insert(index, value) : 처음, 중간에 요소삽입 -> 삽입 후 기존 인덱스 재배치
